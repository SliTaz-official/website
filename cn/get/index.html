镜像源

Slitaz的镜像源在法国、瑞士、中国均有分布。在法国由Botux、Mésobius和TuxFamily提供，在瑞士由SWITCHmirror提供，在中国由LupaWorld提供。感谢这些网站对Slitaz的支持。

  • 瑞士 - SWITCHmirror提供的镜像源 (10 Gbit/s) HTTP FTP
  • 法国 - Mésobius提供的镜像源 (100 Mps) HTTP
  • 法国 - TuxFamily提供的镜像源 HTTP FTP
  • 中国 - LupaWorld提供的镜像源 HTTP
  • 美国 - Ibiblio提供的镜像源 HTTP FTP
  • 巴西 - UFPR提供的镜像源 HTTP FTP

源码

Tazpkg（软件包管理器）、Tazlito（Slitaz Live工具）、Tazwok（软件包制作）、Tazusb（LiveUSB工具）和一些其他的Slitaz小工具可以在Slitaz镜像源里下载。所有小工具Slitaz都默认安装有，源代码的注释是用英文书写的。你可以在镜像源的sources文件夹里找到这些源码。 HTTP FTP

LiveCD - 稳定版

Slitaz稳定版提供一个成熟的操作系统。这个操作系统已经经过严格的测试，基本上没有BUG。LiveCD。这个版本选择了许多有用的软件，是一个用途广泛、强大的操作系统。这个版本较为稳定，不过更新频率较慢。建议服务器用户或追求稳定的个人用户使用这个版本。当前最新的稳定版本是2.0版，在2009年4月16日发布。

SliTaz GNU/Linux 2.0 (29 MB) - slitaz-2.0.iso [ md5 ]

提供给2.0版本的，包含所有软件包的可启动CD镜像每天都会创建。它也包含Slitaz官方网站的副本，在无网络的情况下可以通过install.sh使用。

SliTaz GNU/Linux 2.0 软件包集合 (< 1Gb) - packages-2.0.iso [ md5 ]

LiveCD - 开发版

Slitaz开发版LiveCD是一个在不断开发的版本，也是可以使用的。它包含的软件包比稳定版多，包含的新特性和新改善也很多，更新时间也较快。不过因为更新较快，不稳定因素也多了许多。建议个人用户使用这个版本。

SliTaz GNU/Linux 开发版-20091104 (31,5 Mb) - slitaz-cooking.iso [ md5 ]

提供给开发版的，包含所有软件包的可启动DVD镜像每天都会创建。它也包含Slitaz官方网站的副本，在无网络的情况下可以通过install.sh使用。这个镜像可以安装到USB设备中，不过会格式化U盘。

SliTaz GNU/Linux 开发版软件包集合 (< 2Gb) - packages-cooking.iso [ md5 ]

启动软盘

这个软盘镜像可以启动在旧电脑中的Slitaz，有没有CD-ROM都可以。它可以启动在USB设备、SlitazCD/DVD、硬盘中的Slitaz，还可以通过网络启动。

软盘镜像(1,44 Mb) - 镜像 [ md5 ]
