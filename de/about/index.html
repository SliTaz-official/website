Über Slitaz

Das SliTaz Projekt

SliTaz GNU/Linux ist ein freies und kostenloses Betriebssystem. Es arbeitet vollständig aus dem Arbeitsspeicher und wird von einem externen Speichermedium wie eine CD oder ein USB-Stick gebootet. Die Installation auf der Festplatte ebenfalls ist möglich. SliTaz wird als ISO-Image veröffentlicht. Auf CD gebrannt können Sie von der CD booten. Sobald das System hochgefahren ist, können Sie die CD entfernen und das Laufwerk für andere Zwecke benutzen. SliTaz ist klein, schnell und leichtgewichtig, bietet aber trotzdem eine vollständige, leicht erweiterbare grafische Oberfläche. Sicherheitsupdates werden für die aktuellen Veröffentlichungen angeboten. Sofern Sie die Live-CD gebrauchen (der Live-USB-Stick kann die persönlichen Daten natürlich selber speichern) können Sie das gesamte Home-Verzeichnis auf einem externen Medium sichern. So erhalten Sie sich alle persönlichen Einstellungen und Dokumente für den nächsten Systemstart.

Das SliTaz Projekt bietet kostenlosen technischen Support an. Dazu stehen Ihnen die Mailing Listen (i18n list) sowie die mehrsprachigen Foren (deutsches Forum) zur Verfügung. Für Anregungen, Anfragen und mehr Informationen stehen wir Ihnen auch gerne per Mail zur Verfügung: <devel at slitaz.org>

Übersicht

  • Root-Dateisystem umfasst etwa 80 MB, das ISO-Image weniger als 30 MB.
  • Sofort verwendbarer Webserver, powered by LightTPD mit CGI und PHP-Support.
  • Im Internet surfen mit dem neuen Mozilla Firefox oder mit Retawq in der Konsole.
  • Soundunterstützung mit Alsa mixer, Audioplayer und CD-Ripper und -Encoder.
  • Chat-, Mail- und FTP-Client integriert.
  • SSH-Client and -Server, powered by Dropbear.
  • Datenbank Engine SQLite.
  • Tool zum Generieren einer Live-USB Version.
  • Tools zum Generieren, Editieren und Brennen von ISO-Images auf CD oder DVD.
  • Eleganter Desktop mit JWM (Cooking: Openbox), Xserver XVesa.
  • Weitere, eigens erstellte, nützliche Tools.
  • Mehr als 440 Pakete (Cooking 580), leicht installierbar über einen Paketmanager.
  • Aktive und freundliche Community.

Web sites & i18n

SliTaz's Website gibt es ebenso auf Französisch, Englisch und Portugiesisch. SliTaz ist Teil des "internationalization project". Die deutsche Sprache wird noch nicht vollständig unterstützt. Jeder, der ein Linux-System schon einmal bedient hat und geringe Englischkenntnisse besitzt wird aber mit Slitaz problemlos klarkommen. Im Gegensatz zur französischen und englischen Seite, ist die deutsche Dokumentation vollständig in das Wiki verlegt. Weitere Hilfe auf deutsch erhalten Sie sonst im Forum.
