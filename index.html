Welcome - Bienvenue - Bienvenido - Bem-vindo - Willkommen

SliTaz is an open source and free operating system providing a full featured desktop or server in less than 30 Mb. SliTaz is simple to use, fast and stable. The distribution can be easily used with a LiveCD and extented with the packages manager. SliTaz web site is available in several languages, more information about the internationalization project...
