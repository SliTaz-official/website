SliTaz tools

La boîte à outils SliTaz (page moving in the Cookbook)

SliTaz tools se présente sous forme d'archive reconnue à leur extension en .tar.gz. L'archive est téléchargeable librement, elle contient des scripts utiles à la création de sa version personnalisée de SliTaz, tel qu'un script de création de nouvelles initramfs et images ISO, les Makefile, ou encore les fichiers de configuration contenu dans /etc. Ce paquet a été créé pour accompager le Cookbook (le livre de cuisine de SliTaz) et pour aider les développeurs SliTaz. L'archive est aussi distribuée car elle contient des fichiers qui peuvent être utiles à des individus de type hacker...

Download Tools - Current version 1.7 (20080206)

  • slitaz-tools-1.7.tar.gz
  • http://download.tuxfamily.org/slitaz/sources/tools/

Mercurial repo

Les SliTaz Tools ont leur propore dépôt Mercurial sur le serveur de SliTaz, ils peuvent être cloné via la commande : 

 $ hg clone http://hg.slitaz.org/slitaz-tools/
