Introdução

O desktop padrão do SliTaz é formado por muitos componentes diferentes. JWM (Joe's Window Manager) e LXPanel combinam-se para prover um desktop que seja simples e funcional.

JWM - Joe's Window Manager

O Joe's Windows manager, escrito em C, é rápido, simples, limpo, estável e eficiente. JWM possui uma barra de tarefas, um menu de ícones e um paginador para a administração de desktops virtuais, a barra de tarefas também podem agir como um dock. Além disso é facilmente configurável com um único arquivo de texto que pode mudar o menu, fontes e seus tamanhos e cores diferentes.

Usando e configurando o JWM

O aplicativo Joe's Window Manager é muito rápido. Para ver o menu apenas clique com qualquer botão do mouse em algum lugar do desktop. Você pode redimensionar uma janela pelas extremidades ou cantos, minimizar ou passar de um desktop virtual a outro pelo paginador. Você também tem atalhos de teclado configuráveis para acesso mais rápido às aplicações que você mais usa. No SliTaz o arquivo de configuração do sistema é /etc/jwm/system.jwmrc. Alem deste arquivo, cada usuário pode usar seu próprio arquivo de configuração escondido dentro de ~/.jwmrc. Este é um arquivo de texto que usa sintaxe XML, e pode ser editado com um editor de texto simples - as linhas que começam com: <!-- são comentários que explicam o que cada tag faz.

Para facilitar a customização do desktop, o SliTaz automaticamente copia durante a inicialização da primeira sessão (gráfica) um arquivo de configuração do sistema para o diretório raiz do usuário. Você pode modificar este arquivo diretamente e pode testar sem risco. Para editar com seu editor de texto favorito: 

 $ geany $HOME/.jwmrc &

Para recuperar um arquivo de configuração original, você pode copiar o arquivo de configuração do sistema e pode renomeá-lo para .jwmrc em seu diretório home: 

 $ cp /etc/jwm/system.jwmrc $HOME/.jwmrc

A tag RootMenu corresponde ao menu exibido clicando com um dos (três) botões do mouse. Para adicionar uma categoria, você tem que usar a tag: Menu - ela contém entradas para vários programas. Qualquer entrada no menu do JWM pode ajustar-se em uma linha. Exemplo usando o aplicativo de gerenciamento de imagem GQview: 

<Program icon="gqview.png" label="GQview">gqview</Program>

Ainda há muitas possibilidades para configurar o RootMenu de acordo com os botões do mouse, a escolha de método para mover janelas, criar grupos, etc. O Manual está disponível on-line no website oficial do projeto. Para ver uma lista de opções de linha de comando, apenas digite jwm -h em um terminal.

Personalize seu JWM

Criar seu próprio estilo gráfico com JWM é relativamente rápido e fácil, as tags são claras e os atributos possíveis são determinados nos comentários. Quando estiver preparando seu trabalho, você pode ver suas modificações reiniciando o gerenciador de janela a partir do menu ou pelo comando jwm -restart. No arquivo de configuração, as tags de estilo vem depois do comentário <!-- Visual Styles-->. Para começar, aqui está uma pequena lista das principais tags de estilo com uma curta descrição:

  • Background gerencia o papel de parede. Esta tag suporta os atributos solid, gradient, image or tile para, respectivamente: usar uma cor sólida, criar um gradiente, exibir uma imagem redimencionada ou ladrilhar uma imagem.
  • BorderStyle controla a borda das janelas.
  • TrayStyle controla a barra de tarefas. A barra de tarefas pode, entre outras coisas, ser escondida automaticamente ou só ocupar uma parte da tela com o atributo width (largura).
  • TrayListStyle controla o estilo da lista de janelas abertas no desktop atual.
  • PagerStyle controla o paginador que exibe os diferentes desktops virtuais (4 por padrão).
  • MenuStyle define o estilo do menu.
  • Os ícones são definidos pela tag IconPath, você pode usar seus próprios ícones especificando o caminho completo para o diretório onde eles estão. Note que você pode especificar mais de um caminho, se quiser, pode usar seus próprios ícones e aqueles contidos nos diretórios /usr/share/pixmaps e /usr/share/icons. O SliTaz usa os ícones do tema Tango: tango.freedesktop.org. Para o menu, eles têm 16x16 e são armazenados em /usr/share/icons/Tango. Você pode adicionar, editá-los, apagá-los... Se quiser instalar ícones novos em seu espaço de usuário, nós lhe aconselhamos que use ~/Picture/Icons (definido por padrão) ou o diretório oculto ~/.Icons.

As cores podem ser definidas pelo nome ou número RGB, como #3A4956. Para usar cores no modo gradiente, você tem que especificar as duas cores separadas por dois pontos, exemplo #6C0023:#3E1220. Você pode mudar fontes e seus tamanhos usando a tag Font. Ainda há algumas coisas pequenas que você pode mudar para personalizar seu desktop, como o nome de um item de menu e seu ícone. Antes de reiniciar o JWM com seu novo arquivo de configuração, você pode conferir sua sintaxe usando o comando: jwm -p. Para maiores detalhes, o manual oficial descreve todas as tags, opções e atributos válidos. Você pode vê-los on-line no website do JWM.

LXPanel

LXPanel é parte do projeto LXDE e segue os padrões Freedesktop. Menus são gerados dinamicamente, acrescentando arquivos .desktop ao diretório /usr/share/applications.

*.desktop files

Um único arquivo .desktop pode conter oito linhas com, respectivamente: - o nome (Name), nome genérico, comentário, o comando para executar (Exec), ícone, tipo e categorias Freedesktop. Exemplo de um arquivo .desktop: 

[Desktop Entry]
Name=XTerm
GenericName=Terminal
Comment=Run commands in a shell
Exec=xterm -bg black -fg white -cr orange -sb -bd violet -rightbar
Icon=xterm.png
Type=Application
Categories=Utility;Terminal;

Note que os ícones ficam situados nos diretórios /usr/share/icons ou /usr/share/pixmaps. A entrada Categories (categorias) declara onde o programa se localizará no menu.

O arquivo de configuração do sistema fica situado em /etc/lxpanel e também pode ser armazenado localmente dentro de ~/.config, entretanto é recomendado que você configure o LXPanel em modo gráfico usando a entrada preferences (preferências) no menu principal.

Configurador do LXPanel

O configurador do LXPanel possui 3 abas:

  • General define a posição, tamanho - dinâmico ou fixo -, fundo e propriedades do painel.
  • Plugins lhe permite adicionar, remover, editar e mover plugins pelo painel.
  • Applications lhe permite definir aplicações preferidas como o gerenciador de arquivo, terminal e comando de logout.
