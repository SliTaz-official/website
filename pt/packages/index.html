Pacotes

Esta parte do site permite a você ter acesso a todos os pacotes de programas disponíveis no mirror do SliTaz. As páginas são automaticamente atualizadas por meio de un cron job. Todos os pacotes são instaláveis via o gerenciador de pacotes Tazpkg usando o comando tazpkg get-install nome-do-pacote.

Documentation

O projeto SliTaz fornece documentação detalhada para permitir que você aprenda como instalar e gerenciar pacotes no seu SliTaz distribuição GNU / Linux. Os membros do fórum de apoio irá ajudá-lo em caso de problemas e também é o lugar para pedir um novo pacote.
